ZEOX Documentation

Learn how to use every feature of the ZEOX obfuscation engine.

Getting Started

Paste your Luau script into the Input panel, configure the options in Config, then hit Obfuscate Script. The output will appear in the right panel and can be copied or downloaded.

Protection Options

OptionEffect
Rename Variables Replaces all local identifiers with short, random names.
Protect Globals Excludes Roblox-specific APIs from renaming to prevent runtime errors.
XOR Encrypt Strings Encodes every string literal as an XOR byte array loaded at runtime.
Control Flow Injects opaque predicates and dead branches to confuse decompilers.
Minify Code Removes all comments, whitespace, and newlines for a compact output.

VM Architecture

ModeDescription
NoneStandard AST-level obfuscation, no VM wrapper.
Register VMCompiles to a custom polymorphic register-based virtual machine.
Stack VMClassic stack-based VM, more compatible but slightly less obfuscated.

File Encryption

Any user can encrypt files up to 500 KB per day. You receive a free 100 KB daily gift that resets every 24 hours. Encrypted files are available for download immediately after processing.
